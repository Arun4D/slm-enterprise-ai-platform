# Frontend - SLM Enterprise AI Platform

This frontend is built with React, TypeScript, Vite, and Tailwind CSS. It provides a ChatGPT-style enterprise UI for the platform.

## Features

- Chat interface with streaming response simulation
- Agent selector
- Plugin marketplace placeholder
- Folder picker for log analysis
- File upload support
- ServiceNow ticket lookup stub
- Settings and admin pages
- Secure login placeholder
- Light/dark theme toggle

## Install and run

```bash
cd frontend
npm install
npm run dev
```

The app should be available at `http://localhost:4173`.

## Build

```bash
npm run build
```

## Notes

This implementation is a frontend starter kit for the SLM enterprise platform and can be connected to the backend via the API once the chat endpoints are available.
