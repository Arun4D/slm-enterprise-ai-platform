# Learning Resources & Code References

This guide provides a curated list of official documentations, portals, and learning resources to help developers understand, extend, and master the code and tooling used in this platform.

---

## 🐍 Python & Backend Orchestration
*   **FastAPI Documentation**: [https://fastapi.tiangolo.com/](https://fastapi.tiangolo.com/) (Guide on routing, async handling, and dependency injection).
*   **Pydantic v2 Docs**: [https://docs.pydantic.dev/latest/](https://docs.pydantic.dev/latest/) (Data validation and setting schemas).
*   **Uvicorn Server**: [https://www.uvicorn.org/](https://www.uvicorn.org/) (ASGI server configuration).
*   **Pytest Testing**: [https://docs.pytest.org/](https://docs.pytest.org/) (Testing syntax, fixtures, and parameterization).

---

## 🤖 Ansible Collection & Playbooks
*   **Ansible Documentation**: [https://docs.ansible.com/](https://docs.ansible.com/) (Playbooks, loops, and conditions syntax).
*   **Ansible Collections Index**: [https://docs.ansible.com/projects/ansible/latest/collections/index_module.html](https://docs.ansible.com/projects/ansible/latest/collections/index_module.html) (Builtin and community module reference).
*   **Ansible Galaxy Namespace (Nutanix)**: [https://galaxy.ansible.com/ui/namespaces/nutanix/](https://galaxy.ansible.com/ui/namespaces/nutanix/) (Nutanix modules index).
*   **community.vmware Collection**: [https://docs.ansible.com/ansible/latest/collections/community/vmware/index.html](https://docs.ansible.com/ansible/latest/collections/community/vmware/index.html) (vSphere integrations).

---

## 🏗️ Terraform Providers & Configuration
*   **Terraform Registry (Nutanix Namespace)**: [https://registry.terraform.io/namespaces/nutanix](https://registry.terraform.io/namespaces/nutanix) (Nutanix provider references).
*   **HashiCorp Terraform Documentation**: [https://developer.hashicorp.com/terraform/docs](https://developer.hashicorp.com/terraform/docs) (HCL blocks, variables, resources lifecycle).
*   **Terraform Registry API Reference**: [https://developer.hashicorp.com/terraform/registry/api](https://developer.hashicorp.com/terraform/registry/api) (Registry dynamic search and downloads).

---

## 🧬 Local SLMs (Offline-First)
*   **llama-cpp-python**: [https://llama-cpp-python.readthedocs.io/](https://llama-cpp-python.readthedocs.io/) (Local GGUF models runtime bindings).
*   **Hugging Face Hub**: [https://huggingface.co/docs/hub/index](https://huggingface.co/docs/hub/index) (Models search, Phi-3, and Qwen download paths).
*   **GGUF Specification**: [https://github.com/ggerganov/ggml/blob/master/docs/gguf.md](https://github.com/ggerganov/ggml/blob/master/docs/gguf.md) (Local memory format specifications).
